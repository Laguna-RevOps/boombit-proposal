Frequently Asked Questions

Q: Will I get blocked by websites?

A: Unlikely. We use IP rotation, human-like delays, and proper User-Agent rotation. Blocking is rare (<1%) with proper configuration.

Q: What if CAPTCHA blocks the form?

A: We automatically solve most CAPTCHAs (90%+ success). If a CAPTCHA can't be solved, we log it and move on. Still a 75-85% overall success rate.

Q: Can I customize the message?

A: Yes. Use {{COMPANY_NAME}} as a placeholder - it auto-replaces with each company's name. No AI - your message goes exactly as you write it.

Q: How many forms can I submit per day?

A: 100-150 comfortably in one run. With parallel runs, you can scale to 500+ per day.

Q: Can I integrate with HubSpot?

A: Yes, but it's an add-on. Core system logs results to CSV/JSON. HubSpot sync can be built later if needed.

Q: What if a website changes its form structure?

A: We'll update the code for free (first 30 days). After that, it's on a per-incident basis.

Q: Do I own the code?

A: Yes. You get the full actor code and can modify it if needed.

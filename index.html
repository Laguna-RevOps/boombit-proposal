Frequently Asked Questions

Will websites block this?

Unlikely. We use IP rotation, human-like submission delays, and proper browser headers. Blocking is rare (<1%) with proper configuration. If a site does block, we log it and continue to the next URL.

What if a form uses a CAPTCHA?

Our system automatically solves most CAPTCHAs (90%+ success rate). If a CAPTCHA cannot be solved, we skip that submission and log the attempt. Overall success rate remains 65–75%.

Can I customize the message sent?

Yes. Provide any message you want with optional {{COMPANY_NAME}} placeholder. The message is submitted exactly as you write it—no modification or AI generation.

How do I use the system?

Simple workflow: Create a CSV file with Company Names and URLs → Upload to Apify dashboard → Click "Start Run" → Monitor progress in real-time → Export results as CSV/JSON.

Can I scale to 10,000+ submissions/month?

Yes. The system is built to scale linearly. At 10,000 submissions, monthly costs increase to ~$70–$100. Just add more target URLs and increase concurrent runs.

What happens if a website changes its form structure?

Our system adapts. The form detection engine analyzes field types and structure on each run. If a form changes, the system automatically adjusts—no manual updates needed. The adaptive logic handles form variations across websites.

Do I own the code?

Yes. You receive full source code ownership. The actor is deployed on Apify under your account. You can modify it, extend it, hire another developer to maintain it, or run it independently.

What about follow-ups and CRM integration?

The core system logs all results (status, success/failure reason, contact info). You handle follow-ups manually via your CRM, email tool, or HubSpot. CRM integrations can be added as a future enhancement if needed.
